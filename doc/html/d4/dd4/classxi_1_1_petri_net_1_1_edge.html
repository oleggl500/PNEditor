Detailed Description

template<class TPos, class TTrans, class TEdge>
class xi::PetriNet::Edge< TPos, TTrans, TEdge >

Implementation of edge of Petri net.

Template class Edge contains data of Edge. The data can be different types. Class Edge also stores Position, Transition and direction (Dir) of the Edge
